# ScammerAway

**Authors:** Blinda Hu, Lawrence Ma, Isaiah Marte

**Event:** Brooklyn Hack

**Description:** ScammerAway is an educational tool that teaches people from all age demographics how to combat scams, as well as informing them of the different types of scams that exist out there on the internet (including phone scams). The best way to combat the prevalent and evolving threat of scamming tomorrow, is to start by educating yourself today.
